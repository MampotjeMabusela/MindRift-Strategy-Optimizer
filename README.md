# MindRift Strategy Optimizer

An AI-powered business strategy evaluator built for Poree Consulting Group.

## Features
- Predict strategy alignment using client metrics
- FastAPI backend with scikit-learn model
- React frontend with Tailwind UI
- Dockerized for cloud deployment
- POPIA/GDPR-compliant logging

## Setup
1. Clone repo
2. Run `train_model.py` to generate model
3. Start backend: `uvicorn main:app --reload`
4. Start frontend: `npm start`
